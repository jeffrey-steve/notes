
**Function:**  
- Provides reliable or fast delivery of segments  
- End-to-end process communication

**Key Concepts:**  
- TCP/UDP  
- Ports, segmentation, flow and error control

**Protocols:**  
- TCP, UDP

**Examples:**  
- HTTP (over TCP), VoIP (over UDP)

**Must-Know:**  
- TCP = reliable, ordered  
- UDP = fast, connectionless

**Common Attacks:**  
- TCP SYN flood  
- UDP flood  
- Port scanning  
- Session hijacking
