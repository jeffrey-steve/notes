**Function:**  
- Manages sessions (start, maintain, terminate)  
- Handles session recovery

**Key Concepts:**  
- Dialog control  
- Session checkpoints

**Examples:**  
- NetBIOS, RPC, SMB session management

**Must-Know:**  
- Syncs and maintains connections  
- Less visible, often embedded

**Common Attacks:**  
- Session hijacking  
- Session fixation  
- Exploiting session timeout vulnerabilities
